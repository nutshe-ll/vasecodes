Get Started



First download Visual Studio Code (VS),
by default, Visual Studio Code is installed in C:\users\{username}\AppData\Local\Programs\Microsoft VS Code,
open it and this will appear.






Ignore the recent files, it just shows recent files we open at VS Code. Now let's hop into the lesson. After we open it, go to your File Explorer. where you save your codes in that app,



Now make a folder in the Documents and name it whatever you like!


After that go back to VS Code and click File, then click Open Folder.



After that, open the folder we just created recently. Finish! You just finish the first step to learn HTML.
